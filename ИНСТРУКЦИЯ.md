# 🏥 Atlant Clinic — МедКарта
## Пошаговая инструкция по развертыванию

---

## Что мы разворачиваем

Полноценное веб-приложение с:
- Базой данных (Supabase — PostgreSQL)
- Авторизацией (email + пароль)
- Ролями (админ / врач)
- Автоматической отправкой email при создании записи
- Доступом с любого устройства по URL
- Хостингом на Vercel (бесплатно)

---

## ШАГ 1: Создать аккаунт GitHub (5 мин)

GitHub — это хранилище для кода. Vercel берет код оттуда.

1. Откройте https://github.com/signup
2. Введите email (тот же Gmail)
3. Создайте пароль
4. Выберите username (например: `atlant-clinic`)
5. Подтвердите email

---

## ШАГ 2: Создать проект на Supabase (10 мин)

Supabase — это облачная база данных.

1. Откройте https://supabase.com
2. Нажмите **"Start your project"**
3. Войдите через **GitHub** (аккаунт из шага 1)
4. Нажмите **"New Project"**
5. Заполните:
   - **Name:** `atlant-medkarta`
   - **Database Password:** придумайте сложный пароль (ЗАПИШИТЕ ЕГО!)
   - **Region:** выберите ближайший (Europe West)
6. Нажмите **"Create new project"** — подождите 2 мин

### Сохранить ключи Supabase:
1. Зайдите в **Settings → API**
2. Скопируйте и сохраните:
   - **Project URL** — выглядит как `https://xxxxx.supabase.co`
   - **anon public key** — длинная строка `eyJ...`
   - **service_role key** — ещё длиннее (СЕКРЕТНЫЙ, никому не показывайте)

### Создать таблицы базы данных:
1. Зайдите в **SQL Editor** (левая панель)
2. Нажмите **"New Query"**
3. Вставьте весь SQL из файла `supabase/schema.sql` (из проекта)
4. Нажмите **"Run"**

### Настроить авторизацию:
1. Зайдите в **Authentication → Providers**
2. Убедитесь что **Email** включен
3. Зайдите в **Authentication → Users**
4. Нажмите **"Add user" → "Create new user"**
5. Создайте первого админа:
   - Email: ваш Gmail
   - Password: надежный пароль
6. После создания — зайдите в **SQL Editor** и выполните:
```sql
UPDATE profiles SET role = 'admin' WHERE email = 'ВАШ_EMAIL@gmail.com';
```

## ШАГ 3: Настроить Gmail для отправки писем (10 мин)

Для автоматической отправки email нужен "App Password".

1. Откройте https://myaccount.google.com/security
2. Включите **"2-Step Verification"** (если ещё не включено)
3. После включения — найдите **"App passwords"**
4. Нажмите **"Select app"** → **"Other"** → введите `Atlant МедКарта`
5. Нажмите **"Generate"**
6. Скопируйте 16-символьный пароль (например: `abcd efgh ijkl mnop`)
7. СОХРАНИТЕ его! Он показывается только один раз.

## ШАГ 4: Загрузить код на GitHub (10 мин)

1. Зайдите на https://github.com
2. Нажмите **"+"** → **"New repository"**
3. Name: `atlant-medkarta`
4. Оставьте **Public** или выберите **Private**
5. Нажмите **"Create repository"**

### Загрузить файлы:
Самый простой способ — через браузер:
1. На странице репозитория нажмите **"uploading an existing file"**
2. Перетащите ВСЕ файлы и папки из проекта
3. Нажмите **"Commit changes"**

Или через Git (если установлен):
```bash
cd atlant-medkarta
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/ВАШ_USERNAME/atlant-medkarta.git
git push -u origin main
```

## ШАГ 5: Развернуть на Vercel (10 мин)

1. Откройте https://vercel.com
2. Нажмите **"Sign Up"** → войдите через **GitHub**
3. Нажмите **"Add New..." → "Project"**
4. Найдите `atlant-medkarta` в списке → нажмите **"Import"**
5. В **"Environment Variables"** добавьте:

| Name | Value |
|------|-------|
| `NEXT_PUBLIC_SUPABASE_URL` | URL из шага 2 |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | anon key из шага 2 |
| `SUPABASE_SERVICE_ROLE_KEY` | service_role key из шага 2 |
| `GMAIL_USER` | ваш Gmail (например: clinic@gmail.com) |
| `GMAIL_APP_PASSWORD` | 16-символьный пароль из шага 3 |

6. Нажмите **"Deploy"**
7. Подождите 2-3 мин — Vercel соберет проект
8. Получите URL: `https://atlant-medkarta.vercel.app`

---

## ШАГ 6: Добавить врачей (5 мин)

1. Откройте свой сайт: `https://atlant-medkarta.vercel.app`
2. Войдите под своим email (админ)
3. Зайдите в **Специалисты** → добавьте врачей с их email
4. Для каждого врача создайте аккаунт:
   - Supabase → Authentication → Add user
   - Или они смогут зарегистрироваться сами на сайте

## Как это работает

### Роли:
- **Админ** — видит всё, может добавлять/удалять врачей, пациентов, управлять складом
- **Врач** — видит своих пациентов, свое расписание, может создавать записи

### Автоматические email:
Когда кто-то создает запись на прием, врач автоматически получает на Gmail:
- Имя пациента
- Дата и время
- Тип приема
- Примечания

### Доступ:
- С компьютера: открыть URL в браузере
- С телефона: открыть URL → "Добавить на главный экран" (работает как приложение)
